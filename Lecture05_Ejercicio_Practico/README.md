# Proyecto de Regresión y Clasificación con Dataset de Películas

En este proyecto se utiliza un dataset de películas para aplicar técnicas de **aprendizaje automático supervisado**, específicamente **regresión lineal** y **regresión logística**. En la primera parte, se emplean modelos de **Ridge y Lasso** para predecir el *rating* de las películas a partir de variables numéricas como la duración (*RunTime*) y el número de votos (*Votes*), utilizando *pipelines*, búsqueda aleatoria de hiperparámetros y validación cruzada. En la segunda parte, el problema se transforma en uno de **clasificación binaria**, donde las películas se categorizan como buenas o malas según su calificación, aplicando regresión logística y evaluando el desempeño mediante *accuracy*, *F1-score* y la matriz de confusión. El proyecto incluye limpieza de datos, entrenamiento, evaluación y visualización de resultados.
